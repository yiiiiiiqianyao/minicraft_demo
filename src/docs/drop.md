# drop design

## generate drop item
玩家在破坏方块时，根据方块的类型生成掉落的物品

### 掉落物品的类型
不同的方块会有不同的掉落物品类型，例如：
- 普通方块：掉落物品为该方块的原始物品
- 树方块：掉落物品为树的叶子

### 掉落的粒子特效
当玩家破坏方块时，会生成一个粒子特效，用于表示物品的掉落

### 掉落物品的概率设置
### 挖掘等级
不同的挖掘等级会有不同的掉落物品概率

## active drop item
掉落物品动画的激活

### 激活范围
掉落物品的动画效果只有在玩家与物品之间的距离小于一定阈值时才会激活，目前激活的范围为玩家当前所处的 chunk 或是 nearFourChunk

### 吸收物品
当玩家与掉落物品接触时，会触发吸收物品的动画效果，物品会被玩家吸收并销毁
