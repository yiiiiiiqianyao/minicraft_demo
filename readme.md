# MC Game Demo

## 基础设置
- 进入游戏的时候，设置的开始时间为早上 12 点
- 游戏时间一昼夜为 12 min
### 版本
- experience: https://lf3-static.bytednsdoc.com/obj/eden-cn/vhfuhpxpf/dev/mc/index.html
- version: 0.0.1

## Keyboard
- `w` 向前移动
- `a` 向左移动
- `s` 向后移动
- `d` 向右移动
- `space` 跳跃
- `z` \ `c` 切换工具栏
- `1` \ `2` \ `3` \ `4` \ `5` \ `6` \ `7` \ `8` \ `9` 切换工具栏中的物品
- `q` 丢弃手上拿着的物品

## DEV
- 通过 query 参数开启 dev 模式

### world
- `world=flat` 开启平坦世界模式

### hour
- `hour=12` 设置固定时间为 12 点

### physics_helper
- `physics_helper=1` 开启物理调试模式

### chunk_helper
- `chunk_helper=1` 开启 chunk 调试模式

### esc
- `esc` 切换第三人称视角/调试使用

### v
- `v` 切换视角
